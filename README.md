# 🧠 Deep Learning Image Classifier

A web-based AI application that classifies images using a deep learning model.
The system allows users to upload an image through a web interface and receive a predicted class label generated by a trained model.

This project demonstrates the integration of **deep learning, computer vision, and web development** to build a user-friendly AI application.

---

# 🚀 Features

* Image upload through a simple web interface
* Deep learning based image classification
* Fast prediction using a trained model
* Clean and user-friendly UI
* Flask backend integration
* Deployable as a live AI web application

---

# 🏗️ Project Architecture

User → Upload Image → Flask Backend → Deep Learning Model → Prediction → Display Result

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/pallavi-dhadage/Image-classifier.git
```

Move into the project directory:

```
cd Image-classifier
```

Install dependencies:

```
pip install -r requirements.txt
```

---

# ▶️ Running the Project

Start the Flask application:

```
python app.py
```

Open the browser and go to:

```
http://127.0.0.1:5000
```

Upload an image and the model will predict the class.

---

# 🧠 Model Workflow

1. User uploads an image.
2. Flask server receives the image.
3. Image is preprocessed.
4. The trained deep learning model performs prediction.
5. Predicted label is returned to the web interface.

---

# 📸 Example Workflow

1. Upload an image.
2. The system processes the image.
3. The model predicts the object class.
4. The result is displayed on the webpage.
